# @m4rch/sv

this is a package for file-serving with automatically added live-reload functionality

# usage

```
$ sv -h

  Usage
    $ sv [dir] [options]

  Options
    -p, --port       specify the port  (default 4000)
    -v, --version    Displays current version        
    -h, --help       Displays this message
```
